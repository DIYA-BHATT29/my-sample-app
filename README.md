# My Sample Application

A simple Flask web application with CSS styling and an image, created for the GitHub repository setup task.

## 📋 Prerequisites

- **Python 3.8+** – [Download Python](https://www.python.org/downloads/)
- **Git** – [Download Git](https://git-scm.com/downloads)
- (Optional) **Virtual environment** – recommended to isolate dependencies

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine.
